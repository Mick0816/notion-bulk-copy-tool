# Notion一括コピーツール

Notionデータベースから複数のページを選択し、テキストとして一括コピーできるツールです。

## 機能

- データベースからページを自動取得
- カテゴリやタグでフィルタリング
- 複数ページを選択してテキスト化
- キャッシュ機能で高速動作
- 並列処理による効率的な読み込み

## 使い方

1. Notion APIトークンとデータベースIDを入力
2. フィルタ設定を読み込み
3. 条件を選択してページを読み込み
4. 必要なページを選択してコピー/保存

## 必要な準備

### Notionインテグレーションの作成

1. https://www.notion.so/my-integrations にアクセス
2. 「新しいインテグレーション」を作成
3. APIトークンをコピー
4. 対象のデータベースに接続を許可

### データベースIDの取得

データベースのURLから取得:
```
https://notion.so/xxxxxxxx?v=yyyyyyyy
```
`xxxxxxxx`の部分がデータベースIDです。

## 注意事項

- リレーションプロパティを使用する場合、リレーション先のデータベースにもインテグレーションを接続してください
- APIトークンは安全に管理してください
